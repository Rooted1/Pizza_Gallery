# Intro to Pizza Gallery Project

#### User Stories ####
  * MVP
    A user should be able to
      enter a name
      pick a type of Pizza
      select pizza size
      select cut style
      select crust type
      see the price of each pizza type
      view items in cart
      see price of each selected pizza in cart
      and get a total price for the order

  * Stretch goals
     A user can
      add desserts
      customize Pizza
      add a drink type
      add a drink size
